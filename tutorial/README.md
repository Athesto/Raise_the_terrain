# Tutorials
> Tutorial of SDL2

[SDL2 - Installation][sdl2]

## Using

to use this tutorial you only need to run the `make` command

```bash
make demo1
make demo2
make demo3
```

This demos compiles into an `a.out` file in the directory. just run it and it as an executable file

```
./a.out
```
